Overview · Components

Components

Reusable React components from oclweb3/src/components/. Each card shows its source file and status. Three exemplars are fully documented — those pages are the template for the rest, which currently only link to source.

What's missing: component detail pages for everything unbadged Stub, plus the design-system gaps flagged in the review (concept selector, key-value-pair display, reference & expansion components). If you write a detail page, lift repo-chip.html as your template.
