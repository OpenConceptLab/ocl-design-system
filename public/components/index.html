Components

Reusable React components from oclweb3/src/components/. Each card shows its source file and status. Eight components have full exemplar pages with live React examples and copyable code snippets. The rest link to source.

New: Component pages now use live React + MUI components — the same code as oclweb3. See the Building Pages guide for page-level patterns.

Buttons

Button Partial

Action buttons, icon buttons, split buttons, and the OCL pill pattern. Covers MUI Button, IconButton, ButtonGroup, and the custom Chip-as-Button wrapper. Includes a catalog of additional button variants found across oclweb3, oclmap, and the Zeplin designs.

Action menus

KebabMenu OK

Compact overflow menu (vertical-ellipsis trigger) for grouping secondary and destructive actions in dense surfaces — table rows, workspace toolbars, list items. Includes a decision table for kebab vs. Manage menu.

Pattern-only · Exemplar

WorkspaceToolbar OK

Right-side action cluster for workspace pages. Tiers actions by importance (primary / secondary / tertiary), collapses secondary into the overflow kebab as the viewport narrows.
